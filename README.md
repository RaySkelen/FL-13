This is a collection of small HTML, CSS, and JS tasks. In each task, there is the 'task' folder with the description of the desired result, and in the 'solution' folder, is the code. This was originally a training programm, so in many tasks there are restrictions, which you can find in .docx file of a particular challenge. In general, there are such tasks - 
1. A simple start, markup the given content using HTML only. 
2. Style the given markup with CSS only, so it would fit the example.
3. Create a page with 2 styles (light-mode and dark-mode) using SASS.
4. Create a page using CSS flex-box or grid systems, so it would not be 'broken' until some precise size. 
5. Create a responsive page with 3 break-points (desktop, tablet, and mobile).
6. 2 simple JS fiddles - find a tip amount, and find the middle character of the word.
7. 'Verify' the user for login and password, and show different messages depending on the inputs; Create a simple simulator of a casino roulette.
8. 7 simple functions, e.g. count the amount of presice letters in a given word, or count scores of 10 football games.
9. 11 simple functions, e.g. create an array from the given range of numbers, format a date in the precise way, write your own Array.prototype.map() and Array.prototype.filter().
10. Create a figting 'game' in a browser console.
11. Create a working file tree using JS only.
12. Create a page with the list of books section and book preview section, with the ability to add, edit, and delete items from the list. Everything should render in real time without refreshing the page. 
13. Create a page using Bootstrap and BEM.
14. Create 2 function constructors using OOP.
15. Create Object.prototype.assign method; Create three different classes with methods without using ES6 classes.
16. Render the given data from a server using AJAX; Implement the editing, disposal, and adding of data on a server.
